# 🏫 Django School Management System

A **School Management System** built with **Django** to manage **student** and **teacher records** efficiently.  
It provides a clean, responsive interface with full **CRUD functionality (Create, Read, Update, Delete)** and Django admin integration for backend management.

---

## 🚀 Features

✅ Add, update, and delete **student** and **teacher** details  
✅ Display all records dynamically in a responsive table  
✅ **ModelForm** integration for clean and reusable form handling  
✅ Fully functional **Django Admin Panel** for easy backend control  
✅ Responsive design using **HTML + CSS (Flexbox & Gradients)**  
✅ Organized structure following **Django’s MVT architecture**

---
